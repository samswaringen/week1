How to Make a New Website on Github

Creating a website on Github or upload one externally!

Make Entirely on Github

  1. Create a new Github account or login to an existing one
  2. Choose an existing repository or create a new one
    • If you are creating a new repository, here is what that will look like. Select repo visibility and what files you want the repo to start with.
  3. Select Create new under the add file drop down
  4. Name the document. Make the name have meaning and value to the overall project.
  5. Comment and Commit!

Make on Desktop and upload to Github

  1. Instead of selecting Create new on step 3 of the previous instructions instead select Upload file
  2. Now go back to desktop and create a new folder This will be the container for all files associated with the new website
  3. Create a new text document named index.html
  4. Now open this folder in your code editor. I'm choosing to use VSCode
    • Remember to rename the index file so its recognized as an html document
  5. Now code your website!
  6. Once finished, go back to github and select Add file
  7. Drag and drop your folder onto the area, add notations and commit!
